# Hooks Guide

Recommended hooks to configure in Cursor Settings → Hooks for every project using this Harness + Symphony template.

## Included Hooks

- `agent-review-on-commit.hook` — Forces Agent Review before any commit
- `post-plan-checklist.hook` — Reminds to run Pre-Plan Checklist after planning
- `post-execution-checklist.hook` — Reminds to run Post-Execution Checklist after execution

## How to Enable
1. Go to Cursor Settings → Hooks tab
2. Add each hook with the appropriate trigger
3. Test them on a small change

These hooks help enforce Harness principles automatically so you don't have to remember every time.
